# 🎬 Netflix Data Analysis

## Project Overview

This project analyzes the Netflix Movies and TV Shows dataset using Python, Pandas, and Matplotlib.

The main objective is to perform data cleaning, exploratory data analysis (EDA), and generate business insights through visualizations.

---

## 📂 Dataset

Netflix Movies and TV Shows Dataset

Columns include:
- Show ID
- Type
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre
- Description

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib

---

## 📊 Exploratory Data Analysis (EDA)

The following analyses were performed:

1. Movies vs TV Shows
2. Top 10 Countries Producing Netflix Content
3. Release Year Analysis
4. Ratings Distribution
5. Top 10 Genres
6. Top 10 Directors
7. Top 10 Actors
8. Movie Duration Distribution
9. Content Added by Year

---

## 💡 Business Insights

- Netflix contains more Movies than TV Shows.
- United States has the highest number of Netflix titles.
- TV-MA is the most common content rating.
- International Movies is the most popular genre.
- Rajiv Chilaka has the highest number of titles as a director.
- Anupam Kher appears in the highest number of titles.
- 2018 has the highest number of releases in this dataset.
